thermalCam43.py:
Derived from thermalCam42.py.  
Masking out regions of spurious IR radiation (eg: a room heater).
MultiSocket receiver is an RPC mechanism to adjust min/max and threshold.
WebStreamer, HSV or gray scale detection
sending  data to Dweet and AWS 
outliers removed from people count before averaging.
use mainly with the web streaming application WebStreamer.py on the Raspberry Pi (the client thermalCam41 can run on
    any PC/ cloud)
